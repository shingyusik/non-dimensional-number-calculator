Dimensionless Numbers

물 물리적 현상의 핵심을 파악하는 무차원수 계산기

Reynolds Number

Re = ρvL / μ

유체의 관성력과 점성력의 비를 나타내며, 유동이 층류인지 난류인지 판단하는 지표입니다.

Mach Number

Ma = v / c

유체의 속도와 음속의 비를 나타내며, 압축성 유동의 특성을 결정합니다.

Nusselt Number

Nu = hL / k

대류 열전달과 전도 열전달의 비를 나타내며, 대류가 열전달에 기여하는 정도를 결정합니다.

Prandtl Number

Pr = μc_p / k

운동량 확산과 열 확산의 비를 나타내며, 유체의 물리적 특성만으로 결정됩니다.
