Dimensionless Numbers

물 물리적 현상의 핵심을 파악하는 무차원수 계산기

Reynolds Number

Re = ρvL / μ

유체의 관성력과 점성력의 비를 나타내며, 유동이 층류인지 난류인지 판단하는 지표입니다.

Mach Number

Ma = v / c

유체의 속도와 음속의 비를 나타내며, 압축성 유동의 특성을 결정합니다.

Nusselt Number

Nu = hL / k

대류 열전달과 전도 열전달의 비를 나타내며, 대류가 열전달에 기여하는 정도를 결정합니다.

Prandtl Number

Pr = μc_p / k

운동량 확산과 열 확산의 비를 나타내며, 유체의 물리적 특성만으로 결정됩니다.

Schmidt Number

Sc = μ / (ρD)

점성 확산율과 분자 확산율의 비를 나타내며, 유동 내의 대류 질량전달을 설명합니다.

Peclet Number

Pe = vL / α

유동에 의한 이송(Advection)과 확산(Diffusion)의 비를 나타냅니다.

Strouhal Number

St = fL / v

비상시 유동(Unsteady flow)의 진동 특성을 나타내는 무차원수입니다.

Froude Number

Fr = v / √(gL)

관성력과 중력의 비를 나타내며, 개수로 유동이나 선박 공학에서 중요합니다.

Weber Number

We = ρv²L / σ

관성력과 표면장력의 비를 나타내며, 액적의 분열이나 기포 형성 분석에 사용됩니다.

Knudsen Number

Kn = λ / L

분자 평균 자유 행로와 특성 길이의 비를 나타내며, 연속체 가설의 타당성을 판단합니다.
